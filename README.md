## ⬇️ Installation Guide
### 1. Prerequites
Make sure you have Python 3 installed on your machine. To check if you have Python, run this command:
```bash
python --version
```
If you have python, you should get something like this!
```bash
Python 3.14.0
```
*If you get an error, that means that you do not have Python installed on your machine! Head over and install Python [here](https://www.python.org)!*
### 2. Clone The Repo
Run this command in the terminal to download all the projects at once:
```bash
git clone https://github.com/Fish2Dev/mini-python-projects.git
```
### 3. Navigate to the Repo
Move into the repo's root folder with this command:
```bash
cd mini-python-projects
```

## 🤝 Feedback & Contribution
Since I am still learning, Feedback is incredibly valuable to me! Whether you're a complete beginner or an experienced developer, here's how you can provide feedback:
### 🐛 Found a Bug or Have an Idea? (Open an Issue)
1. Click on the **Issues** tab at the top of this GitHub page.
2. Click the green **New Issue** button.
3. Describe the problem or your idea, and I'll check it out!

### 💻 Want to Squash Some Bugs? (Submit a Pull Request)
If you want to actively improve the code yourself:
1. **Fork** this repository to your own GitHub account.
2. Make your edits or add a new mini-project to its own folder.
3. Submit a **Pull Request (PR)** back to this repository so I can review your changes and merge them into the main project!
* Note: Please do not submit pull requests adding massive new features. We want thse projects to stay simple and beginner-friendly! If you do have an idea for a new feature, please share it as an issue or keep the feature in your own fork.
