# CLI To-Do List App: Project Overview

**Quick Summary**: A CLI app where people can manage their everyday tasks inside a simple to do list.

## ⚡ Installation & Quick Start
```bash
# 1. Navigate to this project's directory
cd todo-list-python

# 2. Run the application
python todo.py

# 3. Go back to the root folder when finished
cd ..
```

## ⚙ Features
* **Feature 1:** Uses a single list (not stored) for tasks.
* **Feature 2:** Add tasks to your list.
* **Feature 3:** Remove and mark tasks as done.
* **Feature 4:** View and clear your list.
* **Feature 5:** Seperates incompleted and completed tasks.

## 🕹️ How to Use & Controls
* **Main Menu:** In the main menu, type the number that correlates to the action you want to do. (ex: select 1 for adding a task)
* **Adding Tasks:** In the main menu, type the number 1 to add a task. Then, type the name of the task.
* **Marking Tasks:** In the main menu, type the number 2 to mark a task as done. Then, type the name of that specific task you want to mark as done (case sensitive).
* **Removing Tasks:** In the main menu, type the number 3 to remove a task. Then, type the name of that specific task you want to remove (case sensitive).
* **Viewing And Clearing The List:** In the main menu, type 4 to view the entire list (seperated by incomplete and complete tasks), and type 5 to clear the entire list.
* **Controls Reference:**
  * 1 - Adding Tasks
  * 2 - Marking tasks
  * 3 - Removing Tasks
  * 4 - View List
  * 5 - Clear List
  * 6 - Quit

## 🛠️ Tech Stack & Concepts
* **Language:** Python 3.x
* **Architecture:** Single File Architecture
* **Data Structures:** In State Memory Tracking (no JSON), Lists.
* **Standard Modules:** sys (for qutting).
* **External Libraries:** None
